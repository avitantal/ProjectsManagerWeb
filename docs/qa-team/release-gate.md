# Release Gate

זהו שער מינימלי לפני פריסה או מיזוג ל-`main`.

## חובה לפני אישור

| Gate | פקודה/בדיקה | נדרש |
| --- | --- | --- |
| Lint | `npm run lint` | Pass |
| Tests | `npm run test:run` | Pass |
| Build | `npm run build` | Pass |
| Product | דרישות רלוונטיות נבדקו | Pass או סיכון מתועד |
| UX | mobile + desktop נבדקו בשינוי UI | Pass או סיכון מתועד |
| Data | פעולות Supabase נבדקו ב-mock או ידנית | Pass או סיכון מתועד |

## חסימות

- Critical פתוח: תמיד חוסם.
- High פתוח: חוסם אלא אם יש החלטה מפורשת של בעל הפרויקט.
- בדיקות אוטומטיות נכשלות: חוסם אלא אם הכשל לא קשור לשינוי ומתועד.
- דרישה לא מתועדת: לא חוסם תיקון קטן, כן חוסם פיצ'ר חדש.

## פלט צפוי

בסיום gate צריך להיות:

- דוח ריצה ב-`.qa/test-runs`.
- דוח סבב ב-`.qa/review-output` כאשר זה סבב ידני.
- רשימת ממצאים פתוחים או החלטה שאין ממצאים.
